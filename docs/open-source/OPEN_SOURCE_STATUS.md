MPL-2.0 is the proposed license only. No license, DCO, public security contact, trademark permission, or governance model has owner/legal approval. The formal Codex Security scan was explicitly deferred and remains mandatory before publication. See `PUBLICATION_GATE_MATRIX.md` and run `python3 script/check_publication_gates.py` for the fail-closed gate result.
